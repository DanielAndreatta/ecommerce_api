Descripción de la WebApp “E-commerce API REST”

Una API REST utilizando DJANGO REST FRAMEWORK, que brinda una funcionalidad básica y acotada de un E-commerce.

ENDPOINTS:

  • CRUD de productos.

    • GET-POST
    api/v1/producto/

    Ruta para listar y crear productos.

    GET-PUT-PATCH-DELETE
    api/v1/producto/{uuid}/

    Ruta para obtener, actualizar y eliminar un producto específico.


  • CRUD de ordenes.

    • GET-POST
    api/v1/orden/

    Ruta para listar y crear órdenes.

    GET-PUT-PATCH-DELETE
    api/v1/orden/{uuid}

    Ruta para obtener, actualizar y eliminar una orden específica.


  • CRUD de detalles de ordenes.

    • GET-POST
    api/v1/detalle_orden/

    Ruta para listar y crear detalles de órdenes.

    GET-PUT-PATCH-DELETE
    api/v1/detalle_orden/{uuid}/

    Ruta para obtener, actualizar y eliminar un detalle de orden específico.


  • ACTUALIZAR stock del producto.

    • PUT-PATCH
    api/v1/producto/{uuid}/stock/

    Ruta para actualizar el campo stock de un producto especifico.


URL del API:

/api/v1/
